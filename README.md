# 🧠 Modern Quiz App

A simple, interactive, and beautifully designed Quiz Application built with pure HTML, CSS, and vanilla JavaScript. This project features a dynamic score tracking system, answer evaluation with instant color-coded feedback, and randomized question shuffling.

## 🚀 Features

- **Dynamic Score Counter**: Tracks your score in real-time as you answer correctly.
- **Randomized Questions**: Questions are shuffled every time the quiz starts or restarts.
- **Instant Visual Feedback**: Buttons change color dynamically (Green for correct, Red for wrong) upon selection.
- **Modern UI/UX**: Designed with a clean slate-dark background, modern typography, elegant cards, and smooth hover effects.
- **Responsive Layout**: Adapts flawlessly to both desktop and mobile screens.

---

## 🛠️ Tech Stack

- **Structure**: HTML5
- **Styling**: CSS3 (Custom Variables, Flexbox, Grid, Media Queries)
- **Logic**: Vanilla JavaScript (ES6+)
